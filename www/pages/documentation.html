Fireman Documentation


Fireman is a Python based responsive firewall manager. It aims to simplify and improve Linux based server-side firewalls by automating firewall packet filtering rules based on the processes running on the server.

Download

To download Fireman, please see our Download page.

Installation

Fireman does not require an install.

Dependencies

To use Fireman, the following dependencies are required:

dicttoxml

Available from the Python Package Index.

xmltodict

Available from the Python Package Index.

python-daemon

Available from the Python Package Index, or through a software package manager, such as Yum.

sudo yum install python-daemon


enum

Available from the Python Package Index, or through a software package manager, such as Yum.

sudo yum install python-enum

Usage

To run fireman, use the following, simple command from the src folder:

python fireman.py
