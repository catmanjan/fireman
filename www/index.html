Project Fireman

Firewall management is hard. Without a security expert onboard it is hard to know if your network is exposed. Fireman aims to fix that.

By automating the firewall process we hope that "security in depth" can be accessible to everyone.

Features

Ease of Use

Simple commands to add and delete rules, or to perform other administrative actions.

Security In Depth

Provide in-depth security by automating your firewall.

Iptables Support

Written using Iptables.

Extensible

Fireman is open source and freely available on Github.
